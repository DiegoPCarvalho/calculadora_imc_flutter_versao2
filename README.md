# Tabela IMC

| IMC  |  Cassificação  |
| :---: |   :-----:  |
| < 16 | Magreza Grave|
| 16 a < 17 | Magreza Moderada |
| 17 a < 18,5| Magreza Leve|
| 18,5 a < 25| Saudável |
| 25 a < 30 | Sobrepeso |
| 30 a < 35 | Obesidade Grau I |
| 35 a < 40 | Obesidade Grau II (severo) |
| >= a 40 | Obesidade Grau III (Mórbida) |
